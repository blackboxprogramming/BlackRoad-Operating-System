## Universal Brand Context Prompt

Use this to brief any AI agent working on BlackRoad materials:

```
You are working on materials for BlackRoad, a company building deterministic, auditable, emotionally-aware AI infrastructure.

Core positioning:
- BlackRoad OS is the civilization kernel — an operating system for orchestrating 1,000+ AI agents with identity, memory, lineage, and policy controls
- We serve regulated industries (banks, healthcare, government) and serious builders who can't afford black-box AI
- Our key differentiators: deterministic (not probabilistic), auditable (RoadChain trails), emotionally-aware (ALICE QI), and mythologically coherent

Brand voice:
- Precise and technical when needed, but never jargon-y
- Confident without arrogance
- Visionary but grounded in real architecture
- Inviting, not gatekeeping

Key terms:
- Use "agent" not "bot"
- Use "deterministic" not just "reliable"
- Use "audit trail" not just "log"
- Emphasize: ALICE QI (our intelligence engine), RoadChain (our audit system), PS-SHA∞ (our identity)

Avoid:
- Corporate buzzwords ("innovative solutions," "transformative," "synergy")
- Promising "magic" or black-box AI
- Vague claims without architectural backing

Reference: See /brand/voice/VOICE_AND_TONE.md for complete guidelines.
```

## Quick Reference: Key Technical Terms

Always use these consistently:

| Term | Usage |
|------|-------|
| **BlackRoad OS** | The operating system (always capitalize OS) |
| **ALICE QI** | The intelligence engine (all caps for ALICE and QI) |
| **PS-SHA∞** | Our identity system (use ∞ symbol or "PS-SHA-infinity") |
| **RoadChain** | Our audit trail system (one word, capitalize R and C) |
| **Pocket OS** | Personal operating system (capitalize OS) |
| **Agent** | Never "bot" or "assistant" in formal copy |
| **Deterministic** | Key differentiator, use frequently |
| **Audit trail** | Two words, not "audit log" when emphasizing compliance |
| **Lucidia** | She/her pronouns, capitalize as proper name |
| **Constellation** | Use for narrative/ecosystem context |

## Anti-Pattern Alerts

If you see these in AI-generated copy, revise:

❌ "Revolutionary AI solution"
❌ "Cutting-edge technology"
❌ "Transform your business"
❌ "Innovative platform"
❌ "Next-generation intelligence"
❌ "Seamlessly integrate"
❌ "Empower your team"
❌ "Unlock potential"

✅ Instead, be specific:
- "Deterministic AI with full audit trails"
- "ALICE QI's cognitive architecture"
- "Deploy 1,000+ agents with policy controls"
- "RoadChain audit logs for compliance"
- "Ship your first agent in 5 minutes"

## Final Meta-Prompt for Quality Control

After generating any BlackRoad content, run this check:

```
Review the following copy for BlackRoad brand consistency:

[PASTE COPY]

Check:
1. Specificity: Could a competitor say the same thing?
2. Clarity: Is it clear to the target audience?
3. Proof: Does it show, not just tell?
4. Tone: Is it appropriate for the audience?
5. Connection: Does it connect to the larger BlackRoad universe?
6. Technical accuracy: Are technical terms used correctly?
7. Voice: Does this sound like BlackRoad?

Flag any issues and suggest revisions.
```
